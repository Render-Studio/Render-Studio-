Render Studio Addon
--------------------------
#### Render Studio For Blender 2.90.
#### This Addon was created on 2 days.

**Q:** What is the Render Studio Addon and that can it do?

**A:** Render Studio is an addon thats allow you to creat a back drop with a 4 way lighting system, 4 cameras way system and eay to use controls.

Camera Controls
--------------------
1. Active Camera
2. Right_Camera Z45
3. Fromt_Camera 90
4. Fromt_Camera X45
5. Left_Camera Z-45
 
Lighting Controls
-----------------------
1. Master_Light
     - Color     
     - Strength
2. Light_Source_Right
    - Color
    - Strength
3. Light_Source_Front
    - Color
    - Strength
4. Light_Source_Left
    - Color
    - Strength

Back Drop Controls
------------------------
1.  Scale
    - Scale X
    - Scale Y
    - Scale Z
2. Shape
    - bevel width_pct
    - bevel segments
    - bevel shape
3. Material
    - distribution
    - subsurface_method
    - Base Color
    - Subsurface
    - Subsurface Radius 
    - Subsurface Color 
    - Metallic
    - Roughness
    - IOR
    - Emission
    - Alpha
