Render-Studio-addon
------------------
### Render Studio For Blender 2.90. 
### This Addon was created on 2 days.

### Q: What is the Render Studio Addon and that can it do?
### A: Render Studio is an addon thats allow you to creat a back drop with a 4 way lighting system, 4 cameras way system and eay to use controls.

Camera Controls
--------------------
- Right_Camera Z45
- Fromt_Camera 90
- Fromt_Camera X45
- Left_Camera Z-45

Lightining Controls
----------------------
- Master_Light
    - Color
    - Strength
- Light_Source_Right
    - Color
    - Strength
- Light_Source_Front
    - Color
    - Strength
- Light_Source_Left
    - Color
    - Strength

Back Drop Controls
-----------------------
- Scale X
- Scale Y
- Scale Z
- bevel width_pct
- bevel segments
- bevel shape
- distribution
- subsurface_method
- Base Color
- Subsurface
- Subsurface Radius
- Subsurface Color
- Metallic
- Roughness
- IOR
- Emission
- Alpha
